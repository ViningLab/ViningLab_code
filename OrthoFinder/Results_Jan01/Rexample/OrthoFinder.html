Species tree

The OrthoFinder species tree describes the relationships among each sample. The species tree includes node labels which are used in other output as well. For example, the Phylogenetic Hierarchical Orthogroups are organized into files based on nodes of the species tree (N0, N1, etc.). This species tree can be read into R and visualized using ape and/or ggtree.

ape

library(ape)
my_tree <- read.tree("../Species_Tree/SpeciesTree_rooted_node_labels.txt")
# my_tree
# my_tree$node.label
my_tree$tip.label
##  [1] "Trema_orientale"                 "Parasponia_andersonii"          
##  [3] "Cascade"                         "Finola_proteins"                
##  [5] "Purple_Kush_proteins"            "W103_proteins"                  
##  [7] "jl_proteins"                     "ERBxHO40_23_hirisehap2_proteins"
##  [9] "HO40_proteins"                   "ERB_F3_proteins"                
## [11] "ERBxHO40_23_hirisehap1_proteins" "Cannbio-2_proteins"             
## [13] "CBDRx"                           "CBDRx_proteins"                 
## [15] "I3_proteins"                     "JL_Mother_proteins"
plot(my_tree)
nodelabels( text = my_tree$node.label, frame = "none", bg = "#FFFFFF", adj = c(1.1, -0.1))
axis(side = 1)

ggtree

suppressMessages(library(ggtree))
library(ggplot2)

#ggtree(my_tree)
p <- ggtree(my_tree) + geom_tiplab() 
# p
p <- p + theme_tree2(plot.margin=margin(16, 30, 16, 30)) +
    scale_x_continuous( limits = c(0, 0.4), breaks = seq(0, 0.4, by = 0.1),
                        labels = seq(0, 0.4, by = 0.1) )
# p

# p + geom_label( aes(x=branch, label = 1:31 ), fill='lightgreen')

p <- p + geom_text(aes(label=c(rep("", times = length(my_tree$tip.label)), my_tree$node.label)), hjust=1.1, vjust=-0.8)

Phylogenetic_Hierarchical_Orthogroups

The OrthoFinder output includes a directory of Phylogenetic Hierarchical Orthogroups. This is a directory of files named by the clades in the species tree. The clade describes which samples have genes in the group and which do not.
